Linear Discriminant Analysis (LDA)
• Assumes a normal distribution
• Selects a new dimension
• Maximizes the separation between classes
• Minimizes the variance within each class after
   projection

Random Forest (RF)
Combines the output of multiple decision trees to reach a single
result.
• Builds many decision trees
• Trains each tree on a random subset of the data
• Uses random subsets of features at each split
• Combines their predictions (majority vote or average)
